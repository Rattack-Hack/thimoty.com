# Installation

Install Nginx/Apache, MariaDB, php and php-{fpm,common,dom,pdo,pdo_mysql,mysql}

Setup a MariaDB database named portfolio_db

Use the nginx.conf at root path for your nginx conf, just adjust root path (for Apache I let you search for the config)

Headers in Nginx conf are necessary for security and for the website to work correctly !

If using Apache enable the following modules :
a2enmod rewrite headers deflate expires

Execute in cli with php the www/setup/generate_admin.php script to generate your admin user and setup the OTP (Yes you have to use OTP for security !).

Replace the example SQL request at end of www/setup/schema.sql by the one generated by the PHP script.

Inject the schema in your database with :
mysql -u root -p portfolio_db < www/setup/schema.sql

Configure your database credentials into config/db.php

There you go you can edit the full website (portfolio/blog) in html via the admin panel https://yoursite.com/dark.php

Delete the www/setup folder for security reasons.
